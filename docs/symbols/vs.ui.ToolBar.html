Class vs.ui.ToolBar

Extends vs.ui.View.

The vs.ui.ToolBar class implements a control for selecting buttons. The vs.ui.ToolBar class provides the ability for the user to customize the tab bar by reordering, removing, and adding items to the bar.

When a tabbar button is selected the tabbar send an event named 'itemselect'. The event data contains the id of the selected button.

configuration
Set a vs.ui.ToolBar configuration

A configuration is an array of button specification.
Q button specification is a array of 2 or 3 values:

  1. The button id (a String)
  2. The button type (a String).
    Ex: vs.ui.ToolBar.BUTTON_BLOG, vs.ui.ToolBar.BUTTON_ARROW_RIGHT
  3. A optional position.
    Ex: vs.ui.ToolBar.LEFT_POSITION or vs.ui.ToolBar.RIGHT_POSITION
    The default value is vs.ui.ToolBar.LEFT_POSITION.
