Class vs.ui.Canvas

Extends vs.ui.View.

The vs.ui.Canvas class is a subclass of vs.ui.View that allows you to easily draw arbitrary content within your HTML content.

When you instantiate the vs.ui.Canvas class you should reimpletement the draw method.
